# 📂 File Organizer Tool

A simple Python tool that automatically organizes your files into folders based on type.

## 🚀 Features
- Organizes files by extension (Images, Docs, Videos, etc.)
- Easy to use
- Lightweight and fast

## 🛠 Requirements
- Python 3.x

## ▶️ How to Use
1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python main.py

3. Select the folder you want to organize
